### 🐑 Installation

```bash
cd project_name
pnpm install
pnpm dev
```

## 🎉 Features

- 🕸️ Site
- 📄 Docs
- ✍️ Blog
  - 🔖 Tags
  - 📟 Pagination
  - 🍕 RSS Feed 2.0 (xml and json)
  - ⏱️ Minutes read time estimation
  - 🧬 Dynamic or provided OG images for blog posts
- 🌎 i18n
- 💬 MDX
- 📍 Sitemap
- 🔍 SEO
- 🌓 Dark/Light mode
- 💅 Tailwind CSS
- 🧱 Shadcn components
- 📐 Responsive
- 🧬 Accessible
- 🤖 Customizable
- 🚀 Open Source

and more...
