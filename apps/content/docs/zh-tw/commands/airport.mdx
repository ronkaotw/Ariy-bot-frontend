`/airport` 指令使用 4 字元 ICAO 代碼查詢機場的詳細資訊。

## 用法

```
/airport <ICAO>
```

| 參數   | 類型   | 必填 | 說明                              |
| ------ | ------ | ---- | --------------------------------- |
| `ICAO` | string | 是   | 4 字元 ICAO 機場代碼（例如 `RCTP`）|

## 回應內容

Bot 會回傳以下摘要：

- 機場名稱與 IATA 代碼
- 城市與國家
- 座標（緯度 / 經度）
- 海拔高度（英尺）
- 機場類型（大型 / 中型 / 小型 / 直升機停機坪）
- 時區

## 範例

```
/airport RCTP
```

**回應：**

```
查詢到的機場為: RCTP - 機場名稱: Taiwan Taoyuan International Airport - 城市名稱: Taoyuan
```

## 備註

- ICAO 代碼為 **4 個字元**（例如 `RCTP`、`VHHH`、`RJTT`、`KLAX`）。
- IATA 代碼（3 個字母，例如 `TPE`）**不被接受**，請務必使用 ICAO 代碼。
- 若找不到機場，Bot 將回傳錯誤訊息。
