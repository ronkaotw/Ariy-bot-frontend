`/start` 指令會傳送歡迎訊息，並列出所有可用指令及其說明。
通常是與 Bot 開啟新對話時傳送的第一個指令。

## 用法

```
/start
```

此指令不接受任何參數。

## 回應

Bot 會回覆：

- 歡迎訊息
- 所有支援指令的清單及簡短說明

## 範例

在 Telegram 或 LINE 傳送 `/start`，Bot 會回覆類似以下內容：

```

哈囉！歡迎使用機師天氣查詢系統！
 以下是所有指令的使用表格：
 /airport ICAO
 /metar ICAO
 /taf ICAO
 /airmet ICAO
 /dev 取得開發人員資料
```

## 備註

- 同時支援 **Telegram** 與 **LINE**。
- 隨時重新傳送 `/start` 可再次顯示說明訊息。
