`/taf` 指令取得指定機場最新的 **TAF**（機場天氣預報）。
TAF 是標準的航空天氣預報格式，涵蓋特定機場未來 24 至 30 小時的天氣狀況。

## 用法

```
/taf <ICAO>
```

| 參數   | 類型   | 必填 | 說明                              |
| ------ | ------ | ---- | --------------------------------- |
| `ICAO` | string | 是   | 4 字元 ICAO 機場代碼（例如 `RCTP`）|

## 回應內容

Bot 會回傳：

- **TAF 原始字串** — 官方發布的編碼預報
- **解碼分段** — 各預報時段的預期狀況，包含：
  - 有效時段（UTC 起訖時間）
  - 預報風向與風速
  - 預期能見度
  - 天氣現象
  - 雲層與天空狀況
  - 變化組（TEMPO / BECMG / PROB，如有）

## 範例

```
/taf RCTP
```

**回應：**

```
TAF: TAF RCTP 311100Z 3112/0118 24010KT 7000 SCT007 BKN011 TEMPO 3112/3116 30005KT FEW012 BKN032 BECMG 3116/3118 01012KT TEMPO 3118/3124 33010KT 4000 SHRA SCT005 BKN008 FEW015CB TEMPO 0100/0104 31012G22KT 3000 TSRA SCT004 BKN006 FEW012CB TEMPO 0104/0107 04010KT FEW010 BKN018 BECMG 0107/0109 07007KT 9999 FEW015 BKN032
```

## 備註

- TAF 有效時段通常為 **24 小時**（部分大型機場發布 30 小時 TAF）。
- TAF 約每 **6 小時**由氣象單位重新發布一次。
- 變化指標說明：
  - **TEMPO** — 短暫波動，每次持續不超過 1 小時
  - **BECMG** — 預期在指定時段內逐漸且永久的改變
  - **PROB30 / PROB40** — 所述狀況發生的機率
- 時間一律以 **UTC（祖魯時間）** 表示。
- 並非所有機場都發布 TAF；小型機場可能無資料回傳。
