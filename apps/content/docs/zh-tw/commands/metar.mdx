`/metar` 指令取得指定機場最新的 **METAR**（機場天氣實況報告）。
METAR 是通報機場地面當前天氣狀況的標準格式。

## 用法

```
/metar <ICAO>
```

| 參數   | 類型   | 必填 | 說明                              |
| ------ | ------ | ---- | --------------------------------- |
| `ICAO` | string | 是   | 4 字元 ICAO 機場代碼（例如 `RCTP`）|

## 回應內容

Bot 會回傳：

- **METAR 原始字串** — 官方發布的編碼報告
- **解碼摘要** — 易讀的說明，包含：
  - 風向與風速
  - 能見度
  - 天氣現象（降雨、霧、雷暴等）
  - 雲層與天空狀況
  - 溫度與露點
  - 高度計設定（QNH）
  - 報告時間（UTC）

## 範例

```
/metar RCTP
```

**回應：**

```
mETAR: METAR RCTP 311230Z VRB01KT 9999 FEW010 SCT015 BKN020 23/22 Q1010 NOSIG RMK A2984
```

## 備註

- METAR 通常每 **30 或 60 分鐘**更新一次。
- 報告反映即時狀況，請注意時間戳記確認資料新鮮度。
- 並非所有機場都發布 METAR；小型機場可能無資料回傳。
- 時間一律以 **UTC（祖魯時間）** 表示。
