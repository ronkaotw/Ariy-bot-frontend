**Ariy Bot** is an aviation weather assistant built for fast, practical airport weather checks.
It helps you query airport data and weather reports directly in chat.

The bot supports Telegram and LINE with one consistent command flow.

## Features

- **Airport lookup**: Query airport details by ICAO code.
- **METAR reports**: Get current weather observations instantly.
- **TAF forecasts**: Check short-term forecast conditions before departure.
- **AIRMET advisories**: Monitor regional weather hazards and alerts.
- **Cross-platform support**: Use the same commands on Telegram and LINE.

## Commands

| Command | Description |
| ------- | ----------- |
| [`/start`](/docs/commands/start) | Display usage help and available commands |
| [`/airport <ICAO>`](/docs/commands/airport) | Get airport information |
| [`/metar <ICAO>`](/docs/commands/metar) | Get current METAR weather observation |
| [`/taf <ICAO>`](/docs/commands/taf) | Get TAF aerodrome forecast |
| [`/airmet <ICAO>`](/docs/commands/airmet) | Get active AIRMET advisories for the region *(unavailable)* |

Example: `/metar RCTP`
